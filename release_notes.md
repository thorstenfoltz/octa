## Features

- Remove maximum column width limit: columns now auto-size without cap and can be resized freely
- Add cell line breaks setting: long text wraps within cells with automatic row height adjustment (Settings > Table View > Cell line breaks)
- Add binary display mode setting: choose between Binary (bits), Hex, or Text (UTF-8) for binary data columns (Settings > Table View > Binary display)
- Add zoom controls: View menu with +/- buttons and percentage display, Ctrl+Plus/Ctrl+Minus to zoom in/out (5% steps, 25%-500%), Ctrl+0 to reset
- Add status bar navigation: type a cell reference (e.g. R5:C3, R5, C3, or a column name) and press Enter to jump directly to that cell
- Show zoom percentage in the status bar when not at 100%
- Binary columns now display actual content (readable text for valid UTF-8, hex or binary digits otherwise) instead of just byte count
